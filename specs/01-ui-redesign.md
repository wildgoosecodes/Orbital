# Orbital UI Redesign Spec

## Style
Futuristic, dark, car-dashboard inspired, clean, minimal.

## Layout
- Left sidebar
- Main dashboard grid
- Top status bar
- AI assistant panel on the right

## Visual Feel
- Rounded cards
- Subtle borders
- Soft glow effects
- Dark gray/black background
- Blue/purple accent colors
- Large readable text

## Keep
- Existing features
- Current component logic

## Change
- Layout
- Spacing
- Colors
- Typography
- Card design

## Do Not Change
- State logic
- File names unless necessary
- App structure unless needed
